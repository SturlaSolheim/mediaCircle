# Media Circle (arbeidstittel)

Dette er et rest api for Media Circle. En applikasjon for roterende medialister

## Hvordan bruke

### Kjør applikasjonen:

go run main.go

### Kjøre tester

For å kjøre alle testene kan du bruke tst.sh scriptet. Det genererer mocks og kjører tester
